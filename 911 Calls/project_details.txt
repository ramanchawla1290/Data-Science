Data Exploration & Visualization on dataset containing information on 911 calls

The data contains the following fields:

* lat : String variable, Latitude
* lng: String variable, Longitude
* desc: String variable, Description of the Emergency Call
* zip: String variable, Zipcode
* title: String variable, Title
* timeStamp: String variable, YYYY-MM-DD HH:MM:SS
* twp: String variable, Township
* addr: String variable, Address
* e: String variable, Dummy variable (always 1)


IMPORTANT NOTE : Project includes Interactive Plots as well, which may not be visible directly on GitHub.
                 Kindly download and run the Jupyter notebook with the required python modules installed.


INCLUDED : Data Subset

Full Dataset : https://www.kaggle.com/mchirico/montcoalert   (Emergency - 911 Calls)


Visualization Libraries Used:

1. MatplotLib
2. Seaborn
3. Plotly (offline) and Cufflinks  (Requires you to run the Jupyter notebook on your computer
                                    with plotly and cufflink modules installed)
